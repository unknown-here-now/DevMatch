# DevMatch

###Description

**DevMatch** is a dynamic networking platform designed for software engineers to connect, collaborate on projects, and bridge the gap between skills and opportunities. Built with a focus on speed and developer experience, it allows users to showcase their tech stacks and find the perfect teammates for their next big idea.

##Tech Stack & Dependencies

### **Frontend**

* **React.js** – UI Library
* **Vite** – Fast Build Tool & Development Server
* **React Router Dom** – Client-side routing
* **Lucide React / FontAwesome** – For iconography

### **Backend & Tools**

* **Node.js** – JavaScript Runtime
* **Express** – Web Framework for the API
* **Dotenv** – Managing Environment Variables
* **Axios** – Promise-based HTTP client for API calls

###Prerequisites

* **Node.js** (Recommended: v18 or higher)
* **npm** or **yarn**
